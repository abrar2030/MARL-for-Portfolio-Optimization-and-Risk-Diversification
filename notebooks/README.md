# Analysis Notebooks

These notebooks provide detailed analysis and visualization:

1. **01_data_exploration.ipynb**: Explore financial data, compute statistics
2. **02_feature_analysis.ipynb**: Analyze feature importance using SHAP
3. **03_model_interpretation.ipynb**: Interpret model decisions
4. **04_regime_analysis.ipynb**: Analyze market regimes

## Usage

```bash
jupyter notebook
```

Then open any of the notebooks to explore the analysis.
